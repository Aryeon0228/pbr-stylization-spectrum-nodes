## 넣기 전 점검

- [ ] 노드 그래프 정리(배치·프레임·dot)를 마친 **최종본**인지 확인
- [ ] 정리 후 SF 출력을 한두 개 다시 렌더링해 기존 결과와 동일한지 확인
- [ ] 패키지에 `Style_Factor` 그래프가 들어 있는지 확인
- [ ] 백업본·테스트본(`*_BACKUP*`, `*_Test*` 등)이 섞여 들어가지 않았는지 확인

4개 파일을 모두 넣은 뒤에는 이 안내 파일(`_README_먼저읽기.md`)을 삭제해도 됩니다.
